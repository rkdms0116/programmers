```
근본적으로 왜 속도차이가 나는지를 잘 모르겠다ㅜㅠ
```

```python
    while stocks:
        # 가장 왼쪽 주식 값 추출
        stock = stocks.popleft()
        time = 0
        # 남아있는 주식들의 가격을 확인하면서
        for price in stocks:
            time += 1
            # 왼쪽 값이 더 큰 순간 멈춤
            if stock > price:
                break
        # for문이 정상적으로 끝나거나 break 걸려도 값 추가
        answer.append(time)
```

💡 마지막에 if문 안에 append를 해줬었는데 생각해보니까 break가 걸려도 for문이 끝나기 때문에 append가 되어서 두 번 들어가는 상황이 발생했었다! 주의하자!
